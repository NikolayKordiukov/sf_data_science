### Описание проекта    
Бизнес-задача: 
Произвести анализ зависимости получения финансирования от некоторых признаков, которые характеризуют НКО
Техническая задача:
Построить модель бинарной классификации НКО (получение финансирования/не получение финансирования) на основе их информации о них.

:arrow_up:[к оглавлению](https://github.com/NikolayKordiukov/sf_data_science/tree/master/project_diplom/README.md#Оглавление)


### Какой кейс решаем?    
Создать модель, предсказыюващую класс НКО (получит финансирование/не получит финансирование), выполнив ряд последовательных этапов и действий в них, а также ответив на вопросы.

**Условия работы:**  
- Предоставляется датасет
- Проводятся основные этапы работы с датасетом
    * Загрузка данных
    * Исследование данных
    * Очистка данных
    * Преобразование признаков
    * Генерация признаков
    * Построение модели
    * Оптимизация модели
    * Подготовка модели к продакшену
- На соответствующем этапе проекта формируются выводы по результатам его выполнения


**Метрика качества**     
- Ответить на вопросы по зависимости получения финансирования от некоторых признаков
- Метрики качества соответствующих моделей

**Что практикуем**     
1. Учимся писать хороший код на python
2. Учимся загружать данные в соответствующем формате
3. Учимся исследовать и преобразовывать исходные данные
4. Учимся генерировать новые признаки и преобразовывать их
5. Учимся отбирать признаки и оценивать их влияние 
6. Учимся строить и оптимизировать модели
7. Учимся строить визуализации
8. Учимся писать комментарии и выводы
